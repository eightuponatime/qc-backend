## commit rules

- **feat**: new feature
- **fix**: bug fix
- **test**: adding or updating tests
- **refactor**: code refactoring
- **docs**: documentation changes
- **build**: build systems or dependencies
- **chore**: maintenance tasks (routine that doesn't change functionality)
